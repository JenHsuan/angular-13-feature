The proxy configuration support ESM (ES6 modules) now. We are able to create the proxy file in the .mjs file format directly.
There are a few steps to set up the proxy.
  1. Create a proxy.conf.mjs file.
  2. Update the angular.json file.
  3. Setup the backend locally (http://localhost:3000).
  4. Start the dev server.
