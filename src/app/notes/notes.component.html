What is APF?
APF can enable a package to work seamlessly under most common scenarios that use Angular.
Because the @angular/core (13.3.0) exports API as APF, we can learn what the APF is by discovering the export section of package.json.
There are a few module types.
  • types: .d.ts files are used by TypeScript
  • esm2020: unflattened ES2022 format
  • es2020: flattened ES2022 format
  • es2015: flattened ES2015 format
  • node: flattened ES2015 format (Node.js 12+ supports ESM)
  • default: flattened ES2022 format
ESM vs FESM
The purpose of ESM (ES2015, ES2020 Module) is to provide a native way to load JavaScript modules into a browser. Before ESM, I have to use third-party library to import JavaScript modules to browsers like require.js (UMD implementation).
In Node.js, we usually import and export JavaScript modules with the common.js format. After Node.js 12, it began to support ESM.
  • ESM: modules are separted in different files individually
  • FESM: flatten all module into a single file

  12. Publish the generated package on npm.
